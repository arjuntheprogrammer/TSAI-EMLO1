About this project

This project is aimed to help Pytorch machine learning developers to quickly build a Flask web app in a Docker container ready to be deployed.
View on GitHub

By default, this app uses MobileNetV2 image classifier that was pre-trained on the ImageNet dataset. This can be easily changed with any custom deep learning model. If you wish to contribute to the project, please feel free to add more example models.
